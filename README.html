Crooked Tree

Live version here. Word press theme designed and devloped for Crooked Tree Coffee House in Dallas, Tx. Key features include :

  • Hambruger drawer navigation
  • Parallax scrolling
  • asynchronous loading

Built using

  • _s as a base theme
  • Bourbon / Neat as a style framework and grid system
  • Sidr for navigation
  • skrollr for scroll based effects and parallax
